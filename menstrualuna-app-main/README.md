# Menstrualuna AI App

## Overview

Menstrualuna is an AI-powered menstrual health tracking and wellness assistant that helps users monitor menstrual cycles, track symptoms, understand health patterns, and receive personalized insights.

## Features

- AI-powered health assistant
- Menstrual cycle tracking
- Symptom monitoring
- Personalized wellness recommendations
- Interactive data visualization
- User-friendly interface

## Tech Stack

- React
- TypeScript
- Vite
- OpenAI API
- Chart.js
- CSS
